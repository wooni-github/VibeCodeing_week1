# Lotto Number Generator

## Overview

A modern, interactive web application that generates random lottery numbers (1-46) with a sequential animation effect and theme support.

## Features

*   **Sequential Animation:** Numbers appear one by one, simulating a real lottery machine.
*   **Theme Toggle:** Switch between Dark Mode and White (Light) Mode.
*   **Dynamic Coloring:** Balls are colored differently based on their value range.
*   **Range:** Generates 6 unique numbers between 1 and 46.

## Technical Details

*   **CSS Variables:** Used for seamless theme switching.
*   **Async/Await:** Used in JavaScript to handle the timing of sequential ball appearances.
*   **Keyframe Animations:** Used for the "pop-in" effect of the lottery balls.
